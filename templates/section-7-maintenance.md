## 7.4 Troubleshooting Guide

| Symptom | Probable Cause | Corrective Action |
|---------|---------------|------------------|
| Machine will not start | E-stop activated | Check and reset all E-stops |
| Machine will not start | Guard open or interlock fault | Check all guards are closed; check interlock status on HMI |
| Pressure loss — pneumatic | Leak in line or fitting | Inspect all fittings; replace damaged components |
| HMI alarm active | See Section 11 for alarm codes | Follow alarm response procedure |
